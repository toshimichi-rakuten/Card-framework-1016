# 楽天カードフレームワーク アイコンフォント仕様書

## 概要

楽天カードフレームワークのアイコンフォントは、UI要素で統一されたアイコンを提供するフォントライブラリです。
175種類のアイコンが利用可能で、全ての楽天カードフレームワークコンポーネントで使用できます。

## 必要な依存関係

### CSS
```html
<link rel="stylesheet" href="https://static.card.jp.rakuten-static.com/card_corp/pc/css/common/iconfont/rakuten-card-icon-5.0.0.min.css">
```

## 基本的な使用方法

### 基本構文
```html
<span class="rakuten-card-icon {icon-name}"></span>
```

### 使用例
```html
<span class="rakuten-card-icon academy-hat"></span>
<span class="rakuten-card-icon check"></span>
<span class="rakuten-card-icon close-button"></span>
```

## 実装時の注意点

### クラス名の構成
- 必ず`rakuten-card-icon`クラスを基底として使用
- アイコン名をクラスとして追加
- 例：`class="rakuten-card-icon check"`

### アクセシビリティ考慮事項
```html
<!-- アイコンのみの場合はaria-labelを追加 -->
<button aria-label="お気に入りに追加">
  <span class="rakuten-card-icon favorite-f"></span>
</button>

<!-- テキストと併用する場合 -->
<button>
  <span class="rakuten-card-icon check"></span>
  確認
</button>

<!-- 装飾的なアイコンの場合はaria-hiddenを追加 -->
<div>
  <span class="rakuten-card-icon info-f" aria-hidden="true"></span>
  重要な情報です
</div>
```

### スタイリング
```css
/* アイコンサイズの調整 */
.rakuten-card-icon {
  font-size: 16px; /* デフォルト */
}

/* 大きなアイコン */
.rakuten-card-icon.large {
  font-size: 24px;
}

/* 色の変更 */
.rakuten-card-icon.primary {
  color: #bf0000; /* 楽天レッド */
}
```

## バージョン互換性

### アイコンフォント 5.0.0
- 全175種類のアイコンが利用可能
- 楽天カードフレームワーク 4.0.0と完全互換

### 旧バージョンからの移行
- アイコン名の変更がある場合があります
- 新規実装では必ず最新バージョンを使用してください
- 既存コードの更新時はアイコン名の確認を行ってください

## トラブルシューティング

### アイコンが表示されない場合
1. CSSファイルの読み込み確認
2. クラス名のスペルチェック
3. ブラウザキャッシュのクリア

### アイコンが正しく表示されない場合
1. フォントの読み込み完了を確認
2. 他のCSSとの競合を確認
3. z-indexやoverflowの設定を確認

### パフォーマンス最適化
- 必要なアイコンのみの読み込み（カスタムビルド）
- フォントプリロードの活用
- CDNの活用による高速化
